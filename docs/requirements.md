## Functional Requirements
1. The system shall allow users to register and log in.
2. The system shall allow users to record income transactions.
3. The system shall allow users to record expense transactions.
4. The system shall generate financial reports.
5. The system shall store all data securely in a database.

## Non-Functional Requirements
1. The system shall be user-friendly and easy to navigate.
2. The system shall respond to user actions within 2 seconds.
3. The system shall ensure data security and access control.
4. The system shall be scalable for future enhancements.
